# Ember & Pine Coffee — Demo Site

A two-page website for **Ember & Pine Coffee**, a fictional coffee shop invented for this project. Built with plain HTML and CSS — no frameworks, no build step.

**Live site:** https://eemersonrosa.github.io/emerson-guimaraes-coffeeshop/

All content is original demo material: the shop, products, menu, and address are made up, and every image is a hand-generated SVG illustration (no stock photography).

## Pages

- **Home** (`index.html`) — hero, product cards, "We're Different" section (hover the can!), and a discount banner.
- **Menu** (`pages/menu.html`) — hot drinks, cold drinks, and pastries with jump navigation.

## Structure

```
├── index.html
├── pages/menu.html
├── css/
│   ├── global.css    # variables, nav, buttons, footer
│   ├── landing.css   # home page
│   └── menu.css      # menu page
└── assets/           # logo, icons, illustrations (all original SVG)
```
